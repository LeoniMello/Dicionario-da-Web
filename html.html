Criando primeiro documento html5

  • <!DOCTYPE html>

    mantido sempre na primeira linha ele serve como uma instrução para definir qual html será usado, no caso atualmente é o html5

  • <html> <html/>

    define onde começa e termina um documento html

  • <head> <head/>

    tag de cabeçalho (não é exibido no conteudo da pagina, carrega apenas as instruções)

  • <body> <body/>

    onde é exibido o conteudo da pagina

  • <meta>

    inserir metadados ao seu documento HTML, por exemplo ao definir <meta charset="UTF-8"> estamos dizendo que iremos usar o conjunto de caracteres UTF-8

  • <!-- comentario -->

    usado para criar comentarios em seu codigo (não é exibido na página)

  • <header> <header/>

    define onde vai ser usado o cabeçalho da pagina
