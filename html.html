• <!DOCTYPE html>

    mantido sempre na primeira linha ele serve como uma instrução para definir qual html será usado, no caso atualmente é o html5

  • <html> <html/>

    define onde começa e termina um documento html

  • <head> <head/>

    tag de cabeçalho (não é exibido no conteudo da pagina, carrega apenas as instruções)

  • <body> <body/>

    onde é exibido o conteudo da pagina
